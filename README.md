## 💫핵심 기능
- 반응형 웹사이트 제작
- firebase 사용하여 로그인/로그아웃, 채용 공고(CRUD) 구현
- react-quill 라이브러리 사용하여 에디터 구현
- firebase storage 파일 다운로드
- 복지몰 상품 무한스크롤 구현
- Recoil을 활용한 장바구니 전역 상태 관리
- Button, InputForm 공통 컴포넌트 관리
